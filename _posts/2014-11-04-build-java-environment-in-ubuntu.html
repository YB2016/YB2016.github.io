一， 安装JDK

其实就是分两步：

1. 下载jdk包，点击进入下载页面，下载完成后找个目录解压，像这种用户程序我习惯放在用户目录下的vhost，也就是/home/slayer/vhost/jdk1.8

2.然后配置环境变量，为什么要配置这个？？这只是为了执行java命令的时候系统能够查找到相关的命令；vim ~/.bashrc加上：

# jdk
export JAVA_HOME=/home/slayer/vhost/jdk1.8
export JRE_HOME=${JAVA_HOME}/jre
export CLASS_PATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

当然这是只对当前用户有效，然后重启终端，执行java -version验证是否配置成功即可。a

或者加入到/etc/profile，再执行source /etc/profile重新加载文件，其他用户也可以使用了。

二，ecplise

我比较喜欢在eclipse下面写java，所以把eclipse装上。

1. 先下载对应的eclipse版本，我的是java se版本，点击进入下载页面。然后解压到一个目录，我的还是放在了/home/slayer/vhost/eclipse/下面;

2. 然后可以从命令行执行./eclipse就可以启动了

3.不过偶尔也会偷懒，为了可以在桌面环境下双击eclipse，我们可以加一个快捷方式到菜单栏，执行

 vim /usr/share/applications/eclipse.desktop

添加以下几行到该文件：

[Desktop Entry]
Type=Application
Name=Eclipse
Comment=Eclipse Integrated Development Environment
Icon=/home/slayer/vhost/eclipse/icon.xpm
Exec=/home/slayer/vhost/eclipse/eclipse
Terminal=false
Categories=Development;IDE;Java;

上面的东西应该很容易看懂，就不多废话了，这样一个简单的java开发环境算是搭建完成了。
